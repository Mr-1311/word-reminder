## **Build**
---
1. `git clone https://github.com/metinUr/word-reminder.git && cd word-reminder`
1. `npm install` or `yarn` 
1. `react-native link` 
1. `react-native run-android` or `react-native run-ios ` 
> **note:** Didn't tested on ios yet.
